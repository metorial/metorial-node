# @metorial/sdk

Primary SDK package for Metorial. Provides the main SDK functionality and session management.

## Installation

```bash
npm install @metorial/sdk
# or
yarn add @metorial/sdk
# or
pnpm add @metorial/sdk
# or
bun add @metorial/sdk
```
